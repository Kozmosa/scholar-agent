TypeScript 类型扩展

新增 TokenUsage interface：
export interface TokenUsage {
  total: {
    input_tokens: number;
    output_tokens: number;
    cache_creation_input_tokens?: number;
    cache_read_input_tokens?: number;
    cost_usd?: number;
  };
  by_model?: Record<string, {
    input_tokens: number;
    output_tokens: number;
    cost_usd?: number;
    ...
  }>;
  source: 'agent-sdk' | 'claude-session-meta';
}
