关键设计决策

决策 选择 理由
EngineEvent 扩展方式 在 EngineEvent 增加可选的 token_usage 字段 不破坏现有 event 结构；token 数据随 system/task_completed 事件携带；也可作为独立 "token" 事件实时发射
Claude Code token 提取 子进程退出后，轮询 session-meta 目录（最多等 30s） session-meta 异步生成；通过 Claude 的内部 session_id 匹配
TaskOutputEvent 暴露 新增 TaskOutputKind.TOKEN = "token"，实时发射 token 快照 SSE 流中可见；前端 MessageStream 可展示 token 进度
SessionService 注入 TaskHarnessService 接收可选的 session_service 引用 最小侵入；task 无 session 时完全跳过
total_cost_usd 聚合 _recalc_session 从 token_usage_json 中 SUM cost_usd 复用现有 recalc 流程；Claude Code 路径无 cost 时跳过
